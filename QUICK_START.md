# Quick Start Guide - Erica Learning Platform

## Prerequisites
- Python 3.9+ installed
- Node.js 18+ installed
- FFmpeg installed (for Manim): `brew install ffmpeg` (macOS) or `sudo apt install ffmpeg` (Linux)

## Installation

### 1. Install Python Dependencies
```bash
cd /Users/aadeechheda/Projects/HackUNCP/erica

# Install Manim system dependencies first (macOS)
brew install py3cairo ffmpeg pango

# Install Python packages
pip install -r requirements.txt
```

### 2. Install Node Dependencies
```bash
npm install
```

## Running the Application

### Start Both Servers

**Terminal 1 - Flask Backend (Port 8000):**
```bash
python app/main.py
```

**Terminal 2 - Next.js Frontend (Port 3001):**
```bash
npm run dev
```

**Access the app:** http://localhost:3001

## Testing the Complete Flow

### 1. Onboarding
- Navigate to http://localhost:3001/onboarding
- Fill in your learning preferences
- Choose **visual** or **auditory** learning (or both)

### 2. Upload Content
- Upload a PDF, PPTX, DOCX, or text file
- Or type in topics like "Photosynthesis" or "Linear Algebra"
- Click "Generate my lesson →"

### 3. Learning Phase
- **Visual learners:** AI-generated Manim animations will play
- **Auditory learners:** AI-generated podcasts will play
- Progress through each module step-by-step

### 4. Knowledge Checks
- After each module, engage with the **AI sparring partner**
- Answer questions in your own words
- Receive intelligent, guiding feedback (not just correct/incorrect)
- The AI helps you think deeper about concepts

### 5. Final Test
- Complete a comprehensive test covering all modules
- Receive feedback only at the end

### 6. Summary
- View your performance statistics
- See what topics you mastered
- Identify areas for improvement
- Access from your account history

## Troubleshooting

### Manim videos not generating?
- Ensure FFmpeg is installed: `ffmpeg -version`
- Check that Cairo and Pango are installed
- Look at Flask console for error messages

### No audio in podcasts?
- Verify ElevenLabs API key in `.env`
- Check your ElevenLabs account quota
- Look for errors in Flask console

### AI sparring partner not responding?
- Verify Gemini API key in `.env`
- Check internet connection
- System will fallback to basic responses if API fails

### Port conflicts?
- Flask runs on **port 8000**
- Next.js runs on **port 3001**
- Make sure these ports are available
